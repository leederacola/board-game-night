Welcome to Board Game Night

What is It?

  • Game-list component & Game-detail component
  • Click a game & get the detailed view
  • Application is hosted with Google Firebase
  • Game data hosted with Firebase Real Time Database

What I want it to be!

  • Board Game Night Scheduler
    • My firends can RSVP to Game Night
    • RSVP'd players game libraries will pooled together
    • Attending Players will vote on available games to play

    Whats up with it?

    • I started with Google Firebase Real Time Database for ease of use to get up and running with an Angular Project
    • Now that my data model is a bit more complex, I have decided to switch to a .Net/SQL backend because thats what employers want in job descriptions.
    • I enjoy Angular so I intend on developing a .Net Core API project so I can continue to use this application
